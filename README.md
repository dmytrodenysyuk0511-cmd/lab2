# UNIChat - Форум та Магазин

Сучасна платформа для спілкування з інтегрованим магазином та системою профілів.

## 🚀 Особливості

### Форум
- 📝 Створення тем та підтем
- 💬 Система коментарів з відповідями
- 📎 Завантаження медіа (фото/відео)
- 🔖 Збереження улюблених постів
- 👤 Публічні профілі користувачів

### Магазин
- 🛍️ Каталог товарів з категоріями
- 🛒 Кошик покупок
- ⭐ Система рейтингів
- 💳 Оформлення замовлень
- 🎁 UNIChat Plus підписка

### Профілі
- 🎨 Персоналізація (аватар, live photo, музика)
- 📊 Статистика активності
- 📜 Історія замовлень
- 🌓 Темна/світла тема
- 🌍 Мультимовність (UA/EN)

## 📦 Встановлення

### Вимоги
- Python 3.10+
- pip

### Крок 1: Клонування репозиторію
```bash
cd C:\Users\dmutr\lab8
```

### Крок 2: Встановлення залежностей
```bash
pip install -r requirements.txt
```

### Крок 3: Застосування міграцій
```bash
python manage.py migrate
```

### Крок 4: Створення суперкористувача (опціонально)
```bash
python manage.py createsuperuser
```

### Крок 5: Збір статичних файлів
```bash
python manage.py collectstatic --noinput
```

### Крок 6: Запуск сервера
```bash
python manage.py runserver
```

Відкрийте браузер: http://127.0.0.1:8000

## 📊 Продуктивність

### До оптимізації
- SQL запити: ~50-100 на сторінку
- Час завантаження: ~800-1200ms
- Розмір сторінки: ~500KB

### Після оптимізації
- SQL запити: ~15-30 на сторінку (-60-70%)
- Час завантаження: ~400-700ms (-40-50%)
- Розмір сторінки: ~450KB (-10%)

## 🛠️ Технології

- **Backend**: Django 6.0.4
- **Database**: SQLite (розробка) / PostgreSQL (production)
- **Cache**: LocMemCache (розробка) / Redis (production)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Стилі**: Custom CSS з анімаціями
